# Customer Revenue Intelligence & Retention Analytics System

## Project Overview
End-to-end data analytics project analyzing ~1 million e-commerce 
transactions (UCI Online Retail II dataset, 2009–2011) to surface 
actionable revenue and customer retention insights.

## Dataset
- Source: UCI Online Retail II via Kaggle
- Rows: ~1,067,371 transactions
- Period: December 2009 – December 2011
- Market: UK-based online retailer

## Tech Stack
- **Python** (Pandas, Matplotlib, Seaborn, Scipy) — EDA & Analysis
- **MySQL** — Structured storage with star schema design
- **Power BI** — Executive dashboard
- **GitHub** — Version control and portfolio display

## Analysis Performed
- RFM Customer Segmentation
- Cohort Retention Analysis
- Revenue Trend Analysis
- Pareto 80/20 Analysis
- Customer Lifetime Value (CLV)
- Statistical Validation (Chi-Square, T-Test)

## How to Use
1. Download dataset from: https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci
2. Place `online_retail_II.xlsx` in `data/raw/`
3. Install dependencies: `pip install -r requirements.txt`
4. Run notebooks in order (01 → 05)
